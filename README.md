## Installation
For user:

    > conda create -n <your_env_name> python=3.10
    > conda activate <your_env_name>
    > pip install formalgeo

For developer:  

    > git clone --depth 1 https://github.com/BitSecret/FormalGeo.git
    > cd FormalGeo
    > conda create -n <your_env_name> python=3.10
    > conda activate <your_env_name>
    > pip install -e .
